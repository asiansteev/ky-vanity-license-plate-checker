# KY Vanity License Plate Checker
Here you will find tools to find all available English words that are currently available as Kentucky vanity license plates.

## Instructions
1. Download words.txt from https://github.com/dwyl/english-words  
2. Run trim_words.rb to remove all words with illegal characters and words over 6 characters.  
3. Run kylicenseplate.rb to check availability of words in new list.
